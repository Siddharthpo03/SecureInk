# SecureInk

Secure document signing platform built with:

- React
- TypeScript
- Express
- Prisma
- PostgreSQL
- Supabase Storage
- JWT Authentication

## Features

- User Registration
- OTP Verification
- Login Authentication
- JWT Authorization
- Document Upload
- Secure PDF Storage
- Document Dashboard
- Secure PDF Viewing

## Upcoming

- Signature Placement
- Multi-Signer Workflow
- Audit Logs
